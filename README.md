<h1>Jogo do número secreto</h1>

<h2>�� Sobre</h2>
O Jogo do Número Secreto é uma brincadeira clássica de adivinhação e lógica. O objetivo principal é descobrir um número que foi escolhido previamente por um jogador ou gerado por um computador.

As regras e formas de jogar variam conforme a versão:

Como jogar (Versão Clássica)
Um participante escolhe um número (por exemplo, entre 1 e 100) e guarda o segredo.

Os outros jogadores dão palpites.

Para cada palpite, quem escolheu o número deve responder apenas com: "É maior", "É menor" ou "Você acertou!".

Vence quem acertar o número com o menor número de tentativas.
